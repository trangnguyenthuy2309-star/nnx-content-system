# Intel Index — xFinance

> Danh sách tập trung toàn bộ intel đã thu thập. Claude cập nhật file này mỗi khi thêm hoặc dùng intel.

---

## Hướng dẫn nhanh

| Lệnh | Kết quả |
|------|---------|
| `"Tìm ý tưởng về [chủ đề]"` | Claude search web → capture → gợi ý carousel |
| `"Xem ý tưởng chưa dùng"` | Hiển thị danh sách status=`new` |
| `"Gợi ý carousel"` | Claude đọc INDEX, đề xuất 3–5 ý tưởng từ intel sẵn có |
| `"Tạo carousel từ [ID]"` | Dùng intel đó làm nguồn cho carousel workflow |
| `"Đánh dấu [ID] đã dùng trong [project]"` | Cập nhật status → `used` |
| `"Archive [ID]"` | Cập nhật status → `archived` |

---

## Status

| Status | Ý nghĩa |
|--------|---------|
| `new` | Chưa dùng — có thể khai thác |
| `used` | Đã dùng trong carousel (xem cột "Dùng trong") |
| `archived` | Hết hạn hoặc không còn phù hợp |
