# Slide Layout — CTA / Summary (Slide cuối)

> Tổng kết hành trình + call-to-action. Slide chốt carousel.

---

```
=== SLIDE LAYOUT: CTA-SUMMARY ===

ROLE: Final slide — recap the journey + drive action (save / share /
follow / read more). Must feel rewarding to viewer who reached this
slide, not abrupt. Closes the loop opened by the Cover.

COMPOSITION ZONES:

ZONE A — Top-left:
  Tag pill: "TỔNG KẾT", "KẾT LUẬN", or "GHI NHỚ".
  Same style as previous slides.

ZONE B — Upper-left (~50% width, ~30% height):
  Display title — 2 lines, second line in terracotta #C9572C.
  Often a question or a definitive statement that closes the loop
  opened by the cover.
  Examples:
    "TỪ TIỀN MẶT ĐẾN" / "NGÂN HÀNG SỐ"
    "5 BÀI HỌC" / "CỐT LÕI"

ZONE C — Below title (~50% width):
  Short tagline (1 line, ≤12 words) with vertical terracotta bar.
  Example: "3 điều đáng nhớ từ lịch sử tài chính"

ZONE D — Middle (full width or split 60/40):
  Recap visual — typically one of:
    (a) "Timeline strip" showing all previous slide concepts in one
        row (small versions of the hero visuals from each slide),
        connected by a flowing terracotta line with small dot markers.
    (b) Three highlighted recap blocks with icons, side-by-side.
    (c) A single hero "destination" visual representing the end state.

ZONE E — Bottom-left (~55% width):
  Three recap points (concise), each with:
    - Small icon (jade #2A8A7F, ~28px)
    - Body text (charcoal) with ONE terracotta key phrase
  Separated by thin dashed dividers.

ZONE F — Bottom-center to bottom-right (~40% width):
  CTA pill: terracotta #C9572C fill, cream text. ONE of:
    - "Lưu lại để xem khi cần"
    - "Theo dõi để xem thêm"
    - "Chia sẻ nếu hữu ích"
  Small bookmark / heart / share icon on the left of pill.

ZONE G — Bottom-left small:
  Optional secondary tagline with lightbulb icon — short, friendly.
  Example: "Hiểu lịch sử để làm chủ tài chính cá nhân"

ZONE H — Bottom-right corner:
  Page counter "N/N" — last number (e.g., "6/6").
  Same style as other slides.

VISUAL HIERARCHY:
1. Display title (closing statement)
2. Recap visual (timeline / three blocks / destination)
3. Three recap points
4. CTA pill
5. Tag + counter

CONTENT FIELDS TO FILL:
- {{TAG_LABEL}} — e.g., "TỔNG KẾT"
- {{TITLE_LINE_1}} / {{TITLE_LINE_2}}
- {{SUBTITLE}}
- {{RECAP_VISUAL_TYPE}} — "timeline strip" | "three blocks" | "destination"
- {{RECAP_VISUAL_DESC}} — English description of what it shows
- {{RECAP_POINT_1}} — Vietnamese, ≤14 words, mark terracotta highlight
- {{RECAP_POINT_2}}
- {{RECAP_POINT_3}}
- {{CTA_LABEL}} — Vietnamese, ≤8 words
- {{CTA_ICON}} — bookmark / heart / share / save
- {{SECONDARY_TAGLINE}} — optional, ≤12 words
- {{SLIDE_NUMBER}} / {{TOTAL_SLIDES}}

CONSISTENCY:
Same as content-bullets — reference the EXACT visual system from
previous slides in the prompt.

=== END SLIDE LAYOUT ===
```
